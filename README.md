# brisk-tools

## 轻松上手

```bash
# npm
npm install brisk-tools

# yarn
yarn add brisk-tools

# pnpm 🔥
pnpm add brisk-tools
```



## 源码获取

```bash
# clone
git clone git@github.com:loveverse/quick-tools.git

# enter
cd brisk-tools

# install
pnpm/npm/yarn install
```
